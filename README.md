# Resumen del Proyecto

Este proyecto es el desarrollo del front-end de una tienda de joyería online. 
Se buscó crear una interfaz de usuario elegante y moderna con una estética minimalista y una paleta de colores dark y sofisticada.

Nota: Todo el diseño UI/UX (incluyendo wireframes y el mockup final) fue conceptualizado y documentado en **Figma**.
